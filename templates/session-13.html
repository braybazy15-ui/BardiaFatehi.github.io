if ساده - تصمیم‌گیری یک‌طرفه

به جلسه سیزدهم خوش اومدی! توی جلسات قبل با عملگرهای مقایسه‌ای آشنا شدی که True یا False برمی‌گردونن. حالا می‌خوایم یاد بگیریم چطور بر اساس این True/False تصمیم‌گیری کنیم.

if اولین و مهم‌ترین ابزار تصمیم‌گیری در برنامه‌نویسیه. با if می‌تونی به برنامه بگی "اگه این شرط درست بود، این کار رو بکن".

بردیا فاتحی

if یعنی چی؟

if یه کلمه کلیدی توی پایتونه که به ما اجازه میده برنامه‌مون تصمیم بگیره.

مثال روزمره: اگه بارون بیاد، چتر برمیدارم.
توی برنامه‌نویسی میشه: if بارون میاد: چتر بردار

if condition: print("شرط درست است")

⚠️ خیلی مهم:

  • بعد از if حتماً : (دو نقطه) میذاریم
  • خط بعدی حتماً فاصله (indent) داره (معمولاً ۴ فاصله)
  • اگه فاصله نذاری، برنامه خطا میده

ساختار if

if شرط: # کدهایی که اگه شرط درست بود اجرا میشن print("شرط برقراره") # می‌تونیم چند خط داشته باشیم x = x + 1 # این خط دیگه داخل if نیست (فاصله نداره)

نکته: هر چی فاصله (indent) داشته باشه، داخل if محسوب میشه. به محض اینکه یه خط فاصله نداشت، if تموم میشه.

مثال ۱: بررسی سن

age = 18 if age >= 18: print("شما بالغ هستید")
خروجی
شما بالغ هستید

مثال ۲: بررسی نمره

score = 15 if score >= 10: print("قبول شدی") print("آفرین!")
خروجی
قبول شدی
آفرین!

اگه شرط false باشه چی؟

اگه شرط if False باشه، هیچ‌کدوم از کدهای داخل if اجرا نمیشن.

age = 15 if age >= 18: print("شما بالغ هستید") print("این خط همیشه اجرا میشه")
خروجی
این خط همیشه اجرا میشه

فقط خط آخر چاپ شد چون داخل if نبود.

مثال عملی با input

age = int(input("چند سالته؟ ")) if age >= 18: print("می‌تونی رای بدی") print("می‌تونی گواهینامه بگیری") print("از مشارکتت ممنونیم")
سن ۲۰
چند سالته؟ 20
می‌تونی رای بدی
می‌تونی گواهینامه بگیری
از مشارکتت ممنونیم
سن ۱۵
چند سالته؟ 15
از مشارکتت ممنونیم

⚠️ اشتباهات رایج:

  • فراموش کردن : بعد از if باید دو نقطه بذاری
  • فراموش کردن فاصله (indent): کدهای داخل if باید فاصله داشته باشن
  • استفاده از = به جای == if x = 5 (غلط) درست: if x == 5
  • تبدیل نکردن input: age = input(...) ← age رو با int تبدیل نکردی

بررسی عدد مثبت

num = int(input("عدد: ")) if num > 0: print("عدد مثبت است")

بررسی زوج بودن

num = int(input("عدد: ")) if num % 2 == 0: print("عدد زوج است")

خلاصه جلسه ۱۳

  • if برای تصمیم‌گیری یک‌طرفه استفاده میشه
  • ساختار: if شرط: (دو نقطه)
  • کدهای داخل if باید فاصله (indent) داشته باشن
  • اگه شرط True باشه، کدهای داخل if اجرا میشن
  • اگه شرط False باشه، هیچ‌کدوم اجرا نمیشن

تمرینات جلسه ۱۳

تمرین ۱:
برنامه‌ای بنویس که سن کاربر رو بگیره و اگه بیشتر از ۱۸ بود، پیام "شما بزرگسال هستید" چاپ کنه.
تمرین ۲:
عددی از کاربر بگیر و اگه مثبت بود، پیام "عدد مثبت است" چاپ کن.
تمرین ۳:
نمره کاربر رو بگیر و اگه بیشتر از ۱۷ بود، پیام "عالی" چاپ کن.
تمرین ۴:
عددی از کاربر بگیر و اگه زوج بود، پیام "زوج است" چاپ کن.
تمرین ۵:
کد زیر مشکل داره. مشکلش چیه؟
age = input("سن: ")
if age >= 18:
    print("بزرگسال")
