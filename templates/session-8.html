تمرین و مرور جلسات ۵-۷

تبریک می‌گم! سه جلسه مهم رو پشت سر گذاشتی: print، متغیرها و input. حالا وقتشه با تمرین کردن، این مفاهیم رو تثبیت کنیم.

توی این جلسه ۱۵ تمرین برات آماده کردم. سعی کن خودت حل کنی، اگه گیر کردی از دکمه "مشاهده جواب" استفاده کن.

بردیا فاتحی

یه مرور سریع

print( )

برای چاپ خروجی

متغیرها

برای ذخیره اطلاعات

input( )

برای گرفتن ورودی

تمرینات ترکیبی (۱۵ تمرین)

تمرین ۱:
برنامه‌ای بنویس که اسمت رو توی یه متغیر ذخیره کنه و با print چاپش کنه.
مشاهده جواب
name = "علی"
print(name)
تمرین ۲:
دو متغیر a و b با مقادیر ۱۰ و ۵ تعریف کن. حاصل جمعشون رو توی متغیر c بریز و چاپ کن.
مشاهده جواب
a = 10
b = 5
c = a + b
print(c)
تمرین ۳:
با استفاده از input، اسم کاربر رو بگیر و چاپ کن "خوش آمدی [اسم]".
مشاهده جواب
name = input("اسمت چیه؟ ")
print("خوش آمدی", name)
تمرین ۴:
با input سن کاربر رو بگیر و بگو "تو [سن] سالته".
مشاهده جواب
age = input("چند سالته؟ ")
print("تو", age, "سالته")
تمرین ۵:
دو عدد از کاربر بگیر (با تبدیل به int) و حاصل جمعشون رو چاپ کن.
مشاهده جواب
a = int(input("عدد اول: "))
b = int(input("عدد دوم: "))
print("جمع:", a + b)
تمرین ۶:
برنامه‌ای بنویس که اسم و سن رو از کاربر بگیره و یه خط چاپ کنه: "سلام [اسم]. تو [سن] سالته."
مشاهده جواب
name = input("اسم: ")
age = input("سن: ")
print("سلام", name, "تو", age, "سالته")
تمرین ۷:
با استفاده از \n، یه شعر دو خطی با یه print چاپ کن.
مشاهده جواب
print("خط اول\nخط دوم")
تمرین ۸:
کد زیر مشکل داره. مشکلش چیه؟ درستش کن.
age = input("سن: ")
print("سال بعد", age + 1)
مشاهده جواب
مشکل: input رشته برمی‌گردونه، باید به int تبدیل کنیم.
age = int(input("سن: "))
print("سال بعد", age + 1)
تمرین ۹:
با یه print، سه خط مختلف رو با استفاده از \n چاپ کن.
مشاهده جواب
print("خط ۱\nخط ۲\nخط ۳")
تمرین ۱۰:
یه متغیر به اسم my_fav تعریف کن و یه چیزی که دوست داری توش بذار. بعد چاپش کن.
مشاهده جواب
my_fav = "پیتزا"
print(my_fav)
تمرین ۱۱:
اسم و فامیل رو با دو تا متغیر جدا از کاربر بگیر و با هم چاپ کن.
مشاهده جواب
first = input("اسم: ")
last = input("فامیل: ")
print(first, last)
تمرین ۱۲:
دو عدد اعشاری از کاربر بگیر (با float) و حاصل ضربشون رو چاپ کن.
مشاهده جواب
x = float(input("عدد اول: "))
y = float(input("عدد دوم: "))
print("ضرب:", x * y)
تمرین ۱۳:
با استفاده از \t، یه جدول ساده دو ستونی از اطلاعات خودت درست کن.
مشاهده جواب
print("اسم:\tعلی")
print("سن:\t25")
print("شهر:\tتهران")
تمرین ۱۴:
سه تا متغیر a, b, c با مقادیر ۵، ۱۰، ۱۵ تعریف کن. حاصل a+b+c رو چاپ کن.
مشاهده جواب
a = 5
b = 10
c = 15
print(a + b + c)
✨ پروژه کوچک:
یه برنامه بنویس که:
۱. اسم رو بگیره
۲. سن رو بگیره
۳. شهر رو بگیره
۴. علاقه‌مندی رو بگیره
۵. آخرش یه معرفی کامل با این فرمت چاپ کنه:
"سلام من [اسم] هستم. [سن] سالمه و اهل [شهر] هستم. به [علاقه‌مندی] علاقه دارم."
مشاهده جواب
name = input("اسمت چیه؟ ")
age = input("چند سالته؟ ")
city = input("کدوم شهری؟ ")
hobby = input("به چی علاقه داری؟ ")
print("سلام من", name, "هستم.", age, "سالمه و اهل", city, "هستم. به", hobby, "علاقه دارم.")
