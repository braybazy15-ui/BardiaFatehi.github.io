حلقه while - تکرار شرطی

به جلسه هفدهم خوش اومدی! تا الان برنامه‌هامون یه بار اجرا میشدن و تموم. اما توی دنیای واقعی، خیلی وقت‌ها نیاز داریم یه کار رو چند بار تکرار کنیم.

مثلاً تا وقتی که رمز رو اشتباه میزنی، دوباره ازت بپرسه. اینجاست که حلقه while به کمکمون میاد.

بردیا فاتحی

حلقه while یعنی چی؟

while یه کلمه کلیدی توی پایتونه که به ما اجازه میده یع قطعه کد رو چند بار تکرار کنیم.

مثال روزمره: تا وقتی که تشنه‌ای، آب بخور.
توی برنامه‌نویسی میشه: while thirsty: drink()

شرط
⬇️
اجرای کدهای حلقه
⬆️
برگشت به شرط
while شرط: # کدهایی که تکرار میشن print("این خط تکرار میشه")

⚠️ خیلی مهم:

  • تا وقتی شرط True باشه، حلقه ادامه پیدا میکنه
  • به محض اینکه شرط False بشه، حلقه تموم میشه
  • باید حواست باشه که شرط بالاخره یه روز False بشه، وگرنه حلقه بی‌نهایت داریم

ساختار while

اجزای حلقه:

  • کلمه کلیدی while
  • یک شرط (مثل x < 5)
  • دو نقطه :
  • کدهای با indent که تکرار میشن

مثال ساده:

counter = 1 while counter <= 3: print("بار", counter) counter = counter + 1
خروجی
بار 1
بار 2
بار 3

چطوری کار میکنه؟
۱. counter=1, شرط 1<=3 هست؟ بله → چاپ "بار 1"، counter میشه 2
۲. counter=2, شرط 2<=3 هست؟ بله → چاپ "بار 2"، counter میشه 3
۳. counter=3, شرط 3<=3 هست؟ بله → چاپ "بار 3"، counter میشه 4
۴. counter=4, شرط 4<=3 هست؟ خیر → حلقه تموم

مثال ۱: اعداد ۱ تا ۵

i = 1 while i <= 5: print(i) i = i + 1
خروجی
1 2 3 4 5

مثال ۲: اعداد زوج ۲ تا ۱۰

i = 2 while i <= 10: print(i) i = i + 2
خروجی
2 4 6 8 10

مثال: تا وقتی رمز درسته

password = "" while password != "secret": password = input("رمز رو وارد کن: ") if password != "secret": print("رمز اشتباهه!") print("ورود موفق!")
نمونه اجرا
رمز رو وارد کن: 123
رمز اشتباهه!
رمز رو وارد کن: admin
رمز اشتباهه!
رمز رو وارد کن: secret
ورود موفق!

⚠️ خطر: حلقه بی‌نهایت (Infinite Loop)

اگه شرط هیچوقت False نشه، حلقه تا ابد ادامه پیدا میکنه و برنامه هنگ میکنه!

i = 1 while i > 0: # همیشه True print(i) i = i + 1 # i همیشه بزرگتر میشه، هیچوقت شرط False نمیشه

✅ راه حل: همیشه مطمئن شو یه جایی شرط False میشه.

مثال ۳: جمع اعداد ۱ تا ۱۰

i = 1 sum_numbers = 0 while i <= 10: sum_numbers = sum_numbers + i i = i + 1 print("جمع:", sum_numbers)
خروجی
جمع: 55

مثال ۴: فاکتوریل

n = 5 factorial = 1 i = 1 while i <= n: factorial = factorial * i i = i + 1 print(f"{n}! = {factorial}")
خروجی
5! = 120

مثال: منو با while

choice = "" while choice != "3": print("\n1. سلام") print("2. وقت") print("3. خروج") choice = input("انتخاب کن: ") if choice == "1": print("سلام دوست من!") elif choice == "2": print("وقت بخیر!") elif choice == "3": print("خدانگهدار!") else: print("انتخاب نامعتبر")

خلاصه جلسه ۱۷

  • while برای تکرار تا وقتی که شرط برقراره استفاده میشه
  • ساختار: while شرط: (با دو نقطه)
  • کدهای داخل حلقه باید indent داشته باشن
  • تا وقتی شرط True هست، حلقه ادامه پیدا میکنه
  • باید مطمئن بشیم شرط بالاخره False میشه (نه حلقه بی‌نهایت)
  • کاربردها: منوها، اعتبارسنجی، شمارنده‌ها

تمرینات جلسه ۱۷

تمرین ۱ (ساده):
با while اعداد ۱ تا ۲۰ رو چاپ کن.
تمرین ۲ (متوسط):
با while اعداد زوج ۲ تا ۵۰ رو چاپ کن.
تمرین ۳ (متوسط):
برنامه‌ای بنویس که مرتب از کاربر عدد بگیره و جمعشون کنه. وقتی کاربر ۰ وارد کرد، برنامه تموم بشه و جمع کل رو چاپ کنه.
تمرین ۴ (چالشی):
برنامه‌ای بنویس که رمز رو از کاربر بگیره و تا وقتی درست وارد نکنه، اجازه نده ادامه بده. (حداکثر ۳ بار اشتباه)
تمرین ۵ (پروژه):
یه منوی رستوران با while بساز که:
1. پیتزا
2. برگر
3. سالاد
4. خروج
کاربر انتخاب کنه و برنامه بر اساس انتخاب، قیمت رو اعلام کنه.
تمرین ۶ (خطایابی):
کد زیر مشکل داره. مشکلش چیه؟ چطور درستش کنیم؟
i = 1
while i < 5:
    print(i)

سوال داری؟
