لیست‌ها (Lists) - معرفی و ایجاد

به جلسه بیست و یکم خوش اومدی! تا الان با متغیرها کار میکردیم که فقط یک مقدار رو ذخیره میکردن. اما توی دنیای واقعی، خیلی وقت‌ها نیاز داریم چندین مقدار رو کنار هم ذخیره کنیم.

مثلاً لیست اسامی دانشجوها، لیست نمرات، لیست قیمت‌ها. اینجاست که لیست‌ها (Lists) به کمکمون میان.

بردیا فاتحی

لیست یعنی چی؟

لیست یه مجموعه مرتب و تغییرپذیر از مقادیر مختلفه. می‌تونیم هر نوع داده‌ای (عدد، رشته لیست دیگه) توی لیست ذخیره کنیم.

مثال روزمره: یه سبد خرید که توش چندتا محصول می‌ذاریم.
توی برنامه‌نویسی میشه: shopping_cart = ["سیب", "موز", "نان"]

# ایجاد یک لیست ساده fruits = ["سیب", "پرتقال", "موز"] numbers = [1, 2, 3, 4, 5] mixed = [10, "سلام", 3.14, True]

روش‌های ایجاد لیست

لیست خالی

my_list = []

لیست بدون عضو

لیست اعداد

nums = [1, 2, 3, 4, 5]

اعداد صحیح

لیست رشته‌ها

names = ["علی", "سارا"]

رشته‌های متنی

لیست ترکیبی

mixed = [1, "text", 3.5]

انواع مختلف

لیست تو در تو

matrix = [[1,2], [3,4]]

لیست داخل لیست

تکرار لیست

zeros = [0] * 5

[0, 0, 0, 0, 0]

دسترسی به اعضا - ایندکس (Index)

هر عضو لیست یه شماره (ایندکس) داره. شماره‌ها از صفر شروع میشن.

fruits = ["سیب", "پرتقال", "موز"] print(fruits[0]) # سیب print(fruits[1]) # پرتقال print(fruits[2]) # موز
خروجی
سیب
پرتقال
موز
# ایندکس منفی (از آخر) print(fruits[-1]) # موز (آخرین) print(fruits[-2]) # پرتقال print(fruits[-3]) # سیب
خروجی
موز
پرتقال
سیب

نکته مهم: اولین عضو ایندکس ۰ داره، دومین عضو ایندکس ۱ و ... ایندکس منفی از آخر شماره میزنه: -1 آخرین، -2 یکی مونده به آخر.

تغییر اعضای لیست

لیست‌ها تغییرپذیر (mutable) هستن. یعنی می‌تونیم اعضاشون رو عوض کنیم.

fruits = ["سیب", "پرتقال", "موز"] print(fruits) fruits[1] = "کیوی" # تغییر عضو دوم print(fruits)
خروجی
['سیب', 'پرتقال', 'موز']
['سیب', 'کیوی', 'موز']

طول لیست - تابع len()

با len( ) می‌تونیم تعداد اعضای لیست رو بفهمیم.

fruits = ["سیب", "پرتقال", "موز", "انگور"] length = len(fruits) print("تعداد میوه‌ها:", length)
خروجی
تعداد میوه‌ها: 4

کاربرد: از len برای پیمایش لیست با حلقه استفاده میکنیم.

عملگر in

بررسی میکنه یه عضو توی لیست هست یا نه.

fruits = ["سیب", "موز", "پرتقال"] print("سیب" in fruits) # True print("کیوی" in fruits) # False

عملگر not in

بررسی میکنه یه عضو توی لیست نیست.

fruits = ["سیب", "موز", "پرتقال"] print("کیوی" not in fruits) # True print("سیب" not in fruits) # False

مثال عملی با input

# گرفتن ۳ عدد از کاربر و ذخیره در لیست numbers = [] for i in range(3): num = int(input(f"عدد {i+1}: ")) numbers.append(num) print("لیست شما:", numbers) print("اولین عدد:", numbers[0]) print("آخرین عدد:", numbers[-1])
نمونه اجرا
عدد 1: 10
عدد 2: 20
عدد 3: 30
لیست شما: [10, 20, 30]
اولین عدد: 10
آخرین عدد: 30

⚠️ خطاهای رایج:

  • IndexError: وقتی به ایندکسی خارج از محدوده دسترسی پیدا کنیم (مثلاً fruits[5] توی لیست ۳ عضوی)
  • TypeError: وقتی از متد اشتباه روی لیست استفاده کنیم
  • فراموش کردن براکت: لیست رو با [] می‌سازیم نه با () یا {}
  • شروع از ۱: یادت باشه ایندکس از ۰ شروع میشه، نه ۱!

خلاصه جلسه ۲۱

  • لیست مجموعه‌ای از مقادیر کنار هم
  • ایجاد: my_list = [1, 2, 3] یا my_list = []
  • دسترسی با ایندکس: my_list[0] (از ۰ شروع میشه)
  • ایندکس منفی: -1 آخرین، -2 یکی مونده به آخر
  • تغییر مقدار: my_list[1] = 10
  • طول لیست: len(my_list)
  • بررسی وجود: item in my_list

تمرینات جلسه ۲۱

تمرین ۱ (ساده):
یه لیست از ۵ تا از میوه‌های مورد علاقت بساز و چاپش کن.
تمرین ۲ (ساده):
با استفاده از ایندکس، اولین و آخرین عضو لیست میوه‌هات رو چاپ کن.
تمرین ۳ (متوسط):
یه لیست از اعداد ۱ تا ۱۰ بساز. عضو سوم رو به ۱۰۰ تغییر بده و لیست جدید رو چاپ کن.
تمرین ۴ (متوسط):
برنامه‌ای بنویس که ۵ عدد از کاربر بگیره و توی یه لیست ذخیره کنه. بعد میانگین اعداد رو حساب کنه.
تمرین ۵ (چالشی):
یه لیست از اسم دوستات بساز. از کاربر یه اسم بگیر و بررسی کن اگه تو لیست بود پیام "پیدا شد" وگرنه "پیدا نشد" چاپ کن.
تمرین ۶ (پروژه):
برنامه‌ای بنویس که یه لیست خالی بسازه. بعد از کاربر مرتب اسم بگیره و به لیست اضافه کنه. وقتی کاربر "exit" رو وارد کرد، برنامه تموم بشه و همه اسم‌ها رو چاپ کنه.

سوال داری؟

اگه مفهوم ایندکس یا تفاوت [] و () رو متوجه نشدی، حتماً بپرس.
