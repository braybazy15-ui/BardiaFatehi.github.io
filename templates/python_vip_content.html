دانلود و نصب پایتون در ویندوز

اولین قدم عملی برای شروع

توی این جلسه میریم سراغ نصب پایتون. قدم به قدم با هم پیش میریم تا پایتون رو روی ویندوز نصب کنیم و آماده بشیم برای نوشتن اولین برنامه‌مون.

جزوه جلسه سوم - نصب پایتون در ویندوز

مراحل نصب پایتون:

  1. وارد سایت python.org بشید
  2. روی دکمه Downloads بزنید
  3. نسخه مخصوص ویندوز رو دانلود کنید
  4. فایل نصب رو اجرا کنید
  5. حتماً تیک گزینه "Add Python to PATH" رو بزنید
  6. روی Install Now کلیک کنید

نکات مهم:

  • حتماً گزینه Add Python to PATH رو فعال کنید
  • اگه خطای نصب داشتید، نصب رو با گزینه "Run as administrator" اجرا کنید
  • بعد از نصب، می‌تونید با دستور python --version توی cmd چک کنید که نصب شده

تست نصب:

برای اطمینان از نصب درست، cmd رو باز کنید و این دستور رو بزنید:

python --version

اگه نسخه پایتون رو نشون داد، یعنی نصب درست بوده.

دانلود و نصب VS Code

معرفی بهترین ویرایشگر کد

حالا که پایتون رو نصب کردیم، نیاز به یه محیط برای نوشتن کدهامون داریم. VS Code بهترین انتخاب برای شروع هستش. توی این جلسه نصبش می‌کنیم و آماده میشیم برای کدنویسی.

جزوه جلسه چهارم - نصب VS Code

چرا VS Code؟

  • رایگان و متن‌باز
  • سبک و سریع

مراحل نصب VS Code:

۵

آموزش دستور print

اولین دستور برنامه نویسی

توی این جلسه با اولین و مهم‌ترین دستور پایتون آشنا میشیم. دستور print به ما کمک میکنه تا خروجی برنامه‌مون رو ببینیم. همچنین با کاراکترهای مخصوص مثل \n آشنا میشیم.

جزوه جلسه پنجم - دستور print

دستور print چیست؟

دستور print در پایتون برای نمایش خروجی استفاده میشه. هر چیزی که داخل پرانتز و بین " " قرار بدیم، توی خروجی نمایش داده میشه.

مثال ساده:

print("سلام دنیا!")

خروجی: سلام دنیا!

چند مثال دیگه:

print("اسم من بردیا است")
print("من دارم برنامه نویسی یاد می‌گیرم")

کاراکتر مخصوص \n چیست؟

\n یه کاراکتر مخصوص هستش که باعث میشه بریم به خط بعدی. بهش میگن new line. 

print("خط اول\nخط دوم")

خروجی:

خط اول
خط دوم

چند مثال با \n:

print("اسم: بردیا\nسن: ۲۰\nشغل: برنامه نویس")

خروجی:

اسم: بردیا
سن: ۲۰
شغل: برنامه نویس

تمرین جلسه:

یه برنامه بنویسید که اسم، سن و علاقه‌مندی‌هاتون رو با استفاده از \n توی سه خط مختلف نمایش بده.

۶

متغیرها در پایتون

ظرف‌هایی برای نگهداری اطلاعات

متغیرها مثل ظرف‌هایی هستن که می‌تونیم اطلاعات مختلف رو توشون ذخیره کنیم. توی این جلسه با انواع متغیرها و نحوه استفاده ازشون آشنا میشیم.

جزوه جلسه ششم - متغیرها

متغیر چیست؟

متغیر یه اسم هستش که به یه مقدار داده میشه. می‌تونیم توی برنامه‌مون ازش استفاده کنیم.

تعریف متغیر:

name = "بردیا"
age = 20
height = 1.85

قوانین نام‌گذاری متغیرها:

  • باید با حرف یا زیرخط (_) شروع بشن
  • نمی‌تونن با عدد شروع بشن
  • فقط می‌تونن شامل حروف، اعداد و زیرخط باشن
  • به حروف بزرگ و کوچک حساس هستن (age و Age فرق دارن)

مثال‌های درست:

my_name = "علی"
age1 = 25
_user = "admin"

مثال‌های نادرست:

1name = "علی"    # غلط
my-name = "علی"  # غلط
class = "ali"    # غلط (کلمه رزرو شده)

نوع داده متغیرها:

  • string (رشته): برای متن‌ها - "سلام"
  • integer (عدد صحیح): برای اعداد - 20
  • float (اعشاری): برای اعداد اعشاری - 1.85
  • boolean (بولی): برای True/False

تمرین جلسه:

سه تا متغیر تعریف کنید: یکی برای اسم، یکی برای سن و یکی برای شهر. بعد با دستور print اونها رو نمایش بدید.

۷

دریافت اطلاعات از کاربر با input

تعامل با کاربر

تا الان برنامه‌هامون فقط خروجی می‌دادن. توی این جلسه یاد می‌گیریم چطور از کاربر اطلاعات بگیریم و برنامه‌های تعاملی بسازیم.

جزوه جلسه هفتم - دستور input

دستور input چیست؟

با دستور input می‌تونیم از کاربر اطلاعات بگیریم. برنامه تا وقتی که کاربر چیزی وارد نکنه، منتظر می‌مونه.

مثال ساده:

name = input("اسمت چیه؟ ")
print("خوش آمدی " + name)

نکته مهم:

هر چیزی که با input دریافت می‌کنیم، از نوع string (رشته) هست. اگه بخوایم با عدد کار کنیم، باید تبدیلش کنیم.

تبدیل به عدد:

age = input("چند سالته؟ ")
age = int(age)  # تبدیل به عدد صحیح
next_year = age + 1
print("سال دیگه میشی " + str(next_year))

تبدیل‌های رایج:

  • int(): تبدیل به عدد صحیح
  • float(): تبدیل به عدد اعشاری
  • str(): تبدیل به رشته

مثال کامل:

name = input("اسمت چیه؟ ")
age = int(input("چند سالته؟ "))
city = input("کدوم شهری؟ ")

print("سلام " + name)
print("تو " + str(age) + " سالته")
print("اهل " + city + " هستی")

تمرین جلسه:

برنامه‌ای بنویسید که دو تا عدد از کاربر بگیره و جمعشون رو نمایش بده.

۸

عملگرهای ریاضی و مقایسه‌ای

انجام محاسبات و مقایسه‌ها

توی این جلسه با عملگرهای ریاضی (جمع، تفریق، ضرب، تقسیم) و عملگرهای مقایسه‌ای (بزرگتر، کوچکتر، مساوی) آشنا میشیم.

جزوه جلسه هشتم - عملگرها

عملگرهای ریاضی:

  • + : جمع
  • - : تفریق
  • * : ضرب
  • / : تقسیم
  • // : تقسیم صحیح
  • % : باقیمانده
  • ** : توان

مثال:

a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.33
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000

عملگرهای مقایسه‌ای:

  • == : مساوی
  • != : نامساوی
  • > : بزرگتر
  • < : کوچکتر
  • >= : بزرگتر یا مساوی
  • <= : کوچکتر یا مساوی

مثال:

x = 5
y = 8

print(x == y)  # False
print(x != y)  # True
print(x > y)   # False
print(x < y)   # True
print(x >= 5)  # True
print(y <= 10) # True

تمرین جلسه:

دو عدد از کاربر بگیرید و تمام عملگرهای ریاضی و مقایسه‌ای رو روی اونها تست کنید.

۹

شرط و if elif else

تصمیم‌گیری در برنامه

توی این جلسه یاد می‌گیریم چطور با استفاده از شرط‌ها، به برنامه‌مون قدرت تصمیم‌گیری بدیم. با if، elif و else می‌تونیم مسیرهای مختلفی برای اجرای برنامه تعریف کنیم.

جزوه جلسه نهم - شرط‌ها

ساختار if:

age = 18

if age >= 18:
    print("شما می‌تونید رای بدید")

ساختار if else:

age = 15

if age >= 18:
    print("شما می‌تونید رای بدید")
else:
    print("شما هنوز نمی‌تونید رای بدید")

ساختار if elif else:

score = 85

if score >= 90:
    print("نمره عالی")
elif score >= 70:
    print("نمره خوب")
elif score >= 50:
    print("نمره متوسط")
else:
    print("نمره ضعیف")

نکات مهم:

  • بعد از if، شرط رو مینویسیم و بعد دو نقطه (:) میذاریم
  • کدهایی که باید اجرا بشن، باید با فاصله (indentation) نوشته بشن
  • میتونیم چند تا elif داشته باشیم
  • else اختیاری هست و آخر همه نوشته میشه

تمرین جلسه:

برنامه‌ای بنویسید که یه عدد از کاربر بگیره و بگه مثبت، منفی یا صفر هست.

۱۰

حلقه for در پایتون

تکرار هوشمندانه با حلقه for

حلقه‌ها به ما اجازه می‌دن یه کار رو چند بار تکرار کنیم. توی این جلسه با حلقه for آشنا میشیم که برای تکرار روی مجموعه‌ها و محدوده‌ها خیلی کاربردیه.

جزوه جلسه دهم - حلقه for

حلقه for چیست؟

حلقه for برای پیمایش روی مجموعه‌ها (مثل رشته‌ها، لیست‌ها) و محدوده‌های عددی استفاده میشه.

ساختار پایه for:

for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

مثال با range:

# از 0 تا 4
for i in range(5):
    print(i)

# از 2 تا 6
for i in range(2, 7):
    print(i)

# از 1 تا 10 با قدم 2
for i in range(1, 11, 2):
    print(i)  # 1, 3, 5, 7, 9

پیمایش روی رشته:

name = "بردیا"
for char in name:
    print(char)

محاسبه مجموع اعداد:

sum = 0
for i in range(1, 11):
    sum += i
print("مجموع اعداد 1 تا 10:", sum)  # 55

جدول ضرب:

number = 5
for i in range(1, 11):
    result = number * i
    print(f"{number} x {i} = {result}")

نکات مهم:

  • range(stop): از 0 تا stop-1
  • range(start, stop): از start تا stop-1
  • range(start, stop, step): با قدم مشخص
  • متغیر حلقه (اینجا i) هر بار مقدار جدید می‌گیرد

تمرین جلسه:

برنامه‌ای بنویسید که اعداد 1 تا 20 را چاپ کند، اما فقط اعداد زوج را نمایش دهد.

۱۱

حلقه while

تا وقتی که شرط برقراره تکرار کن

حلقه while تا زمانی که شرط ما درست باشه، به تکرار ادامه میده. این حلقه وقتی مفیده که تعداد تکرار از قبل مشخص نیست.

جزوه جلسه یازدهم - حلقه while

حلقه while چیست؟

حلقه while تا زمانی که شرط مشخص شده True باشه، به تکرار ادامه میده. به محض اینکه شرط False بشه، حلقه تموم میشه.

ساختار پایه while:

counter = 0
while counter < 5:
    print(counter)
    counter += 1

مثال: دریافت عدد از کاربر تا وقتی که صفر وارد کنه

number = int(input("عدد وارد کن (0 برای خروج): "))
while number != 0:
    print("شما وارد کردید:", number)
    number = int(input("عدد وارد کن (0 برای خروج): "))

مثال: محاسبه فاکتوریل

n = 5
factorial = 1
i = 1
while i <= n:
    factorial *= i
    i += 1
print(f"فاکتوریل {n} = {factorial}")  # 120

مثال: حدس عدد

secret = 7
guess = 0
while guess != secret:
    guess = int(input("عددی بین 1 تا 10 حدس بزن: "))
    if guess < secret:
        print("برو بالا!")
    elif guess > secret:
        print("بیا پایین!")
print("آفرین! درست حدس زدی")

تفاوت for و while:

  • for: وقتی تعداد تکرار مشخصه یا روی مجموعه می‌خوایم پیمایش کنیم
  • while: وقتی به شرط وابسته‌ایم و تعداد تکرار از قبل مشخص نیست

هشدار:

مواظب حلقه بی‌نهایت باشید! همیشه مطمئن شوید که شرط حلقه بالاخره یه روز False میشه.

تمرین جلسه:

برنامه‌ای بنویسید که اعداد 1 تا 10 را با while چاپ کند.

۱۲

لیست‌ها در پایتون

مجموعه‌های مرتب از داده‌ها

لیست‌ها یکی از پرکاربردترین ساختارهای داده در پایتون هستن. با لیست‌ها می‌تونیم چندین مقدار رو در یک متغیر ذخیره کنیم و به راحتی مدیریتشون کنیم.

جزوه جلسه دوازدهم - لیست‌ها

لیست چیست؟

لیست یه مجموعه مرتب و تغییرپذیر از مقادیر مختلفه. می‌تونیم انواع داده‌ها رو توی یه لیست ذخیره کنیم.

ایجاد لیست:

# لیست خالی
my_list = []

# لیست با مقادیر
numbers = [1, 2, 3, 4, 5]
fruits = ["سیب", "پرتقال", "موز"]
mixed = [1, "سلام", 3.14, True]

دسترسی به اعضای لیست (ایندکس):

fruits = ["سیب", "پرتقال", "موز"]
print(fruits[0])  # سیب
print(fruits[1])  # پرتقال
print(fruits[-1]) # موز (آخرین عضو)

تغییر مقادیر لیست:

fruits[1] = "کیوی"
print(fruits)  # ["سیب", "کیوی", "موز"]

پیمایش لیست با for:

for fruit in fruits:
    print(fruit)

متدهای مهم لیست:

  • append(): اضافه کردن به انتهای لیست
    • fruits.append("انگور")
  • insert(): اضافه کردن در جای مشخص
    • fruits.insert(1, "انار")
  • remove(): حذف اولین مقدار مشخص شده
    • fruits.remove("کیوی")
  • pop(): حذف و برگردوندن آخرین عضو
    • last = fruits.pop()
  • len(): تعداد اعضای لیست
    • print(len(fruits))

مثال: میانگین اعداد

numbers = [15, 20, 35, 40, 50]
sum = 0
for num in numbers:
    sum += num
average = sum / len(numbers)
print("میانگین:", average)  # 32.0

تمرین جلسه:

یه لیست از ۵ عدد مختلف بسازید و بزرگترین عدد رو پیدا کنید.

۱۳

تابع در پایتون

بازنویسی کدها با توابع

توابع به ما اجازه میدن یه قطعه کد رو یک بار بنویسیم و هر جا نیاز داشتیم ازش استفاده کنیم. اینطوری برنامه‌مون خواناتر و قابل استفاده‌تر میشه.

جزوه جلسه سیزدهم - تابع

تابع چیست؟

تابع یه بلوک از کد هست که یه کار مشخص رو انجام میده. می‌تونیم بهش ورودی بدیم و ازش خروجی بگیریم.

تعریف تابع ساده:

def say_hello():
    print("سلام!")
    print("به پایتون خوش آمدی")

صدا زدن تابع:

say_hello()  # تابع اجرا میشه

تابع با ورودی:

def greet(name):
    print(f"سلام {name} عزیز!")

greet("علی")  # سلام علی عزیز!

تابع با چند ورودی:

def add(a, b):
    result = a + b
    print(f"جمع {a} و {b} = {result}")

add(5, 3)  # جمع 5 و 3 = 8

تابع با خروجی (return):

def multiply(x, y):
    return x * y

result = multiply(4, 5)
print(result)  # 20

مثال کاربردی:

def is_even(number):
    if number % 2 == 0:
        return True
    else:
        return False

num = 7
if is_even(num):
    print(f"{num} زوج است")
else:
    print(f"{num} فرد است")

تابع با مقدار پیش‌فرض:

def power(base, exponent=2):
    return base ** exponent

print(power(3))     # 9 (3 به توان 2)
print(power(3, 4))  # 81 (3 به توان 4)

مزایای استفاده از تابع:

  • کاهش تکرار کد (DRY - Don't Repeat Yourself)
  • خوانایی بیشتر برنامه
  • اشکال‌زدایی آسان‌تر
  • قابلیت استفاده مجدد

تمرین جلسه:

تابعی بنویسید که یه عدد بگیره و تشخیص بده اول هست یا نه.

تبریک می‌گم! شما موفق شدید

به خانواده بزرگ برنامه‌نویسان پایتون خوش آمدید!

اما این فقط شروع مسیر هست. دنیای برنامه‌نویسی خیلی بزرگتر از این حرفاست و شما الان ابزارهای لازم برای ادامه رو دارید.

ادامه مسیر رو چطور ادامه بدیم؟

مهم‌ترین قدم الآن اینه که بدونید هدف شما از برنامه‌نویسی چیه؟ چون بر اساس هدف‌تون، مسیر ادامه یادگیری متفاوت میشه. بیاید با هم نگاه کنیم به حوزه‌های مختلف:

توسعه وب

هوش مصنوعی

برای علاقه‌مندان به هوش مصنوعی، کتابخانه‌هایی مثل TensorFlow و PyTorch رو باید یاد بگیرید.

علم داده

اگه به کار با داده‌ها و تحلیل علاقه دارید، کتابخانه‌های Pandas، NumPy و Matplotlib رو شروع کنید.

اتوماسیون

برای خودکارسازی کارهای روزمره و نوشتن اسکریپت‌های کاربردی، این مسیر عالیه.

چند توصیه مهم برای ادامه مسیر:

  • هدف خود را مشخص کنید: ببینید به کدوم حوزه علاقه بیشتری دارید
  • پروژه محور یاد بگیرید: بعد از یادگیری هر مبحث، یه پروژه کوچیک انجام بدید
  • تمرین روزانه: حتی روزی ۲۰ دقیقه کد بزنید تا مهارتتون حفظ بشه
  • توی جامعه برنامه‌نویسی عضو بشید: از تجربیات دیگران استفاده کنید
  • پروژه‌های خود را در گیت‌هاب قرار بدید: رزومه شما برای کارفرماها
  • هیچوقت یادگیری رو متوقف نکنید: تکنولوژی همیشه در حال تغییره

یادگیری برنامه‌نویسی مثل دوچرخه‌سواری می‌مونه. اولش سخته، زمین می‌خوری، اما بعدش دیگه فراموش نمی‌کنی و می‌تونی ازش لذت ببری.
