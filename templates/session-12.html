پروژه: ماشین حساب ساده

تبریک میگم! به اولین پروژه عملی رسیدی! توی جلسات قبل با عملگرهای ریاضی آشنا شدی و توی جلسه ۱۱ کلی تمرین کردی. حالا وقتشه که اولین پروژه واقعی خودت رو بنویسی.

این پروژه ماشین حساب هست. شاید ساده به نظر بیاد، اما تمام چیزهایی که تا الان یاد گرفتی رو جمع میکنه.

بردیا فاتحی

مرحله ۱: مرور داشته‌ها

برای ساختن ماشین حساب، به این چیزها نیاز داریم:

✅ از جلسه ۵
  • print برای خروجی
  • کاراکترهای خاص مثل \n
✅ از جلسه ۶
  • متغیرها
  • انواع داده (int, float)
✅ از جلسه ۷
  • input برای گرفتن اعداد
  • تبدیل نوع (int/float)
✅ از جلسه ۹
  • عملگرهای ریاضی
  • اولویت عملگرها

ساخت ماشین حساب قدم به قدم

قدم ۱: گرفتن اعداد از کاربر
# دو عدد از کاربر می‌گیریم num1 = float(input("عدد اول را وارد کنید: ")) num2 = float(input("عدد دوم را وارد کنید: "))

از float استفاده کردیم تا کاربر بتونه عدد اعشاری هم وارد کنه.

قدم ۲: انجام عملیات ریاضی
# متغیرهای مختلف برای هر عملیات sum_result = num1 + num2 sub_result = num1 - num2 mul_result = num1 * num2 div_result = num1 / num2 floor_div = num1 // num2 remainder = num1 % num2 power = num1 ** num2
قدم ۳: نمایش نتایج
# نمایش با خط جداکننده print("\n" + "="*40) print("نتایج محاسبات:") print("="*40) print(f"جمع: {num1} + {num2} = {sum_result}") print(f"تفریق: {num1} - {num2} = {sub_result}") print(f"ضرب: {num1} * {num2} = {mul_result}") print(f"تقسیم: {num1} / {num2} = {div_result}") print(f"تقسیم صحیح: {num1} // {num2} = {floor_div}") print(f"باقیمانده: {num1} % {num2} = {remainder}") print(f"توان: {num1} ** {num2} = {power}") print("="*40)

کد نهایی ماشین حساب

# ماشین حساب ساده - پروژه جلسه ۱۲ # آکادمی برنامه نویسی بردیا فاتحی # گرفتن اعداد از کاربر num1 = float(input("عدد اول را وارد کنید: ")) num2 = float(input("عدد دوم را وارد کنید: ")) # انجام عملیات sum_result = num1 + num2 sub_result = num1 - num2 mul_result = num1 * num2 div_result = num1 / num2 floor_div = num1 // num2 remainder = num1 % num2 power = num1 ** num2 # نمایش نتایج print("\n" + "="*40) print("نتایج محاسبات:") print("="*40) print(f"جمع: {num1} + {num2} = {sum_result}") print(f"تفریق: {num1} - {num2} = {sub_result}") print(f"ضرب: {num1} * {num2} = {mul_result}") print(f"تقسیم: {num1} / {num2} = {div_result}") print(f"تقسیم صحیح: {num1} // {num2} = {floor_div}") print(f"باقیمانده: {num1} % {num2} = {remainder}") print(f"توان: {num1} ** {num2} = {power}") print("="*40)

نکات مهم پروژه

  • از float استفاده کردیم تا اعشار هم کار کنه
  • خط جداکننده (===) باعث میشه خروجی مرتب‌تر بشه
  • از f-string برای نمایش مرتب استفاده کردیم
  • برنامه رو با اعداد مختلف تست کن

خطاهای احتمالی

  • ZeroDivisionError: اگه عدد دوم رو ۰ بدی، تقسیم و باقیمانده خطا میدن
  • ValueError: اگه به جای عدد، حرف وارد کنی
  • عدد خیلی بزرگ: توان اعداد بزرگ نتیجه خیلی بزرگی میده

تمرینات اضافه (اختیاری)

✨ تمرین ۱: ماشین حساب با اعشار

برنامه رو طوری تغییر بده که فقط دو رقم اعشار نشون بده. (راهنمایی: از {result:.2f} استفاده کن)

✨ تمرین ۲: ماشین حساب هوشمند

اگه عدد دوم صفر بود، به جای خطا، پیغام "تقسیم بر صفر ممکن نیست" چاپ کن.

✨ تمرین ۳: ماشین حساب سه عددی

برنامه رو طوری تغییر بده که سه عدد از کاربر بگیره و عملیات رو روی هر سه انجام بده.

✨ تمرین ۴: ماشین حساب با منو

(پروژه پیشرفته) یه منو بساز که کاربر انتخاب کنه کدوم عملیات انجام بشه.
