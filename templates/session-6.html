متغیرها در پایتون (Variables) - ظرف‌های اطلاعات

به جلسه ششم خوش اومدی! توی جلسه قبل با print آشنا شدیم و فهمیدیم چطور خروجی بگیریم. حالا می‌خوایم بریم سراغ یکی از مهم‌ترین مفاهیم برنامه‌نویسی یعنی متغیرها.

متغیرها مثل جعبه‌هایی می‌مونن که می‌تونیم هر چی خواستیم توشون ذخیره کنیم.

بردیا فاتحی - مدرس و منتور شما

متغیر (Variable) یعنی چی؟ مثال روزمره

بیا با یه مثال ساده شروع کنیم. فرض کن یه جعبه داری که می‌تونی هر چی خواستی توش بذاری. روی جعبه یه برچسب می‌زنی تا بدونی توش چیه.

مثال جعبه:
  • جعبه با برچسب "سن" → توش عدد ۲۵ گذاشتم
  • جعبه با برچسب "نام" → توش متن "علی" گذاشتم
  • جعبه با برچسب "قد" → توش عدد ۱.۸۵ گذاشتم

توی برنامه‌نویسی به این جعبه‌ها می‌گیم متغیر. به برچسب جعبه می‌گیم اسم متغیر و به چیزی که توش می‌ذاریم می‌گیم مقدار متغیر.

چطور توی پایتون متغیر تعریف کنیم؟

توی پایتون، تعریف متغیر خیلی سادست. فقط کافیه یه اسم انتخاب کنی، بعد = بزاری و بعد مقدار مورد نظرت رو بدی.

name = "علی" age = 25 height = 1.85 is_student = True
حالا چاپشون کنیم:
print(name) print(age) print(height) print(is_student)
خروجی
علی
25
1.85
True

نکته مهم: وقتی متغیر رو چاپ می‌کنی، دیگه نیازی به " " نیست. چون می‌خوای مقدار داخل متغیر رو ببینی، نه یه متن ثابت رو.

انواع داده (Data Types) در پایتون

String (رشته)

برای متن‌ها

name = "علی"
Integer (عدد صحیح)

برای اعداد بدون اعشار

age = 25
Float (اعشاری)

برای اعداد اعشاری

height = 1.85
Boolean (بولی)

True یا False

is_student = True
NoneType

هیچی، خالی

x = None

قوانین درست

  • با حرف یا زیرخط (_) شروع بشه
  • فقط شامل حرف، عدد، _ باشه
  • به حروف بزرگ و کوچک حساسه
  • معنی‌دار باشه (مثلاً age نه a)
name = "علی" # درست user_name = "ali" # درست age1 = 25 # درست

قوانین نادرست

  • با عدد شروع نشه
  • (فاصله) نداشته باشه
  • کلمات کلیدی نباشه (if, for, ...)
  • کاراکتر خاص (@, #, $) نداشته باشه
1name = "علی" # غلط user name = "ali" # غلط if = 25 # غلط

تغییر مقدار متغیرها

متغیرها تغییرپذیر هستن. یعنی می‌تونیم هر لحظه که خواستیم، مقدار جدیدی توشون بذاریم.

x = 10 print(x) # 10 x = 20 print(x) # 20 x = "سلام" print(x) # سلام

نکته جالب: توی پایتون، متغیرها می‌تونن نوع داده خودشون رو عوض کنن! یه لحظه عدد بودن، لحظه بعد شدن متن.

مثال ۱: اطلاعات شخصی

name = "محمد" age = 30 city = "تهران" print("اسم:", name) print("سن:", age) print("شهر:", city)

مثال ۲: عملیات ریاضی

a = 10 b = 3 sum = a + b print("جمع:", sum) product = a * b print("ضرب:", product)

⚠️ خطاهای رایج:

  • NameError: وقتی متغیری رو که تعریف نکردی، صدا بزنی
  • SyntaxError: وقتی اسم متغیر رو اشتباه بنویسی (مثلاً با عدد شروع کنی)
  • TypeError: وقتی عدد و متن رو اشتباهی با + جمع بزنی

خلاصه جلسه ششم

  • متغیر یعنی یه ظرف برای نگهداری اطلاعات
  • تعریف: name = "علی"
  • انواع داده: string, integer, float, boolean
  • اسم متغیر باید با حرف یا _ شروع بشه
  • متغیرها می‌تونن مقدارشون عوض بشه
  • به حروف بزرگ و کوچک حساسیم (age و Age فرق دارن)

تمرین‌های جلسه ششم

تمرین ۱ (ساده):
سه تا متغیر تعریف کن: name (اسم خودت)، age (سن خودت)، city (شهر خودت) و هر سه رو با print چاپ کن.
تمرین ۲ (متوسط):
دو تا متغیر عددی a و b تعریف کن. بعد حاصل جمع، تفریق، ضرب و تقسیمشون رو توی متغیرهای جداگانه بریز و چاپ کن.
تمرین ۳ (قانون نام‌گذاری):
کدوم از اسم‌های زیر برای متغیر درست هستن؟
- 1name
- user_name
- first name
- _private
- class
- firstName
تمرین ۴ (عملی):
یه متغیر به اسم x با مقدار ۱۰ تعریف کن. بعد مقدارش رو به ۲۰ تغییر بده و چاپ کن. بعد به "سلام" تغییر بده و چاپ کن.
تمرین ۵ (چالشی):
یه برنامه بنویس که با استفاده از متغیرها، یه معرفی کوتاه از خودت به این شکل چاپ کنه:
"سلام، من [نام] هستم. [سن] سالمه و اهل [شهر] هستم."
تمرین ۶ (تحقیقی):
تحقیق کن ببین type( ) توی پایتون چه کاری انجام میده. با یه مثال تستش کن.
