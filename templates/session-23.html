بازگشت به لیست جلسات
۲۳

پیمایش لیست با حلقه

به جلسه بیست و سوم خوش اومدی! توی دو جلسه قبل با لیست‌ها و متدهای کار باهاشون آشنا شدیم. حالا می‌خوایم یاد بگیریم چطور روی لیست‌ها حرکت کنیم و با تک تک اعضاشون کار کنیم.

پیمایش لیست با حلقه، یکی از قدرتمندترین تکنیک‌ها تو برنامه‌نویسیه. با این کار می‌تونیم جمع اعداد رو حساب کنیم، چیزی رو توی لیست جستجو کنیم، یا لیست جدیدی بسازیم.

بردیا فاتحی

پیمایش ساده با حلقه for

ساده‌ترین راه برای پیمایش لیست، استفاده از حلقه for هست.

fruits = ["سیب", "موز", "پرتقال", "انگور"] for fruit in fruits: print(fruit)
خروجی
سیب
موز
پرتقال
انگور

توضیح: متغیر fruit در هر بار اجرای حلقه، یکی از اعضای لیست رو می‌گیره و توی بدنه حلقه می‌تونیم باهاش کار کنیم.

پیمایش با ایندکس

گاهی نیاز داریم به شماره (ایندکس) اعضا هم دسترسی داشته باشیم.

fruits = ["سیب", "موز", "پرتقال"] for i in range(len(fruits)): print(f"عضو {i}: {fruits[i]}")
خروجی
عضو 0: سیب
عضو 1: موز
عضو 2: پرتقال
# روش دوم: استفاده از enumerate for i, fruit in enumerate(fruits): print(f"شماره {i}: {fruit}")
خروجی
شماره 0: سیب
شماره 1: موز
شماره 2: پرتقال

enumerate: این تابع هم ایندکس و هم مقدار رو برمی‌گردونه.

مثال‌های کاربردی

مجموع اعداد
numbers = [10, 20, 30, 40] total = 0 for num in numbers: total += num print("مجموع:", total)
خروجی
مجموع: 100
میانگین
numbers = [15, 20, 25, 30] total = 0 for num in numbers: total += num average = total / len(numbers) print("میانگین:", average)
خروجی
میانگین: 22.5
پیدا کردن بزرگترین
numbers = [45, 12, 78, 34, 56] max_num = numbers[0] for num in numbers: if num > max_num: max_num = num print("بزرگترین:", max_num)
خروجی
بزرگترین: 78
شمارش زوج‌ها
numbers = [1, 2, 3, 4, 5, 6] even_count = 0 for num in numbers: if num % 2 == 0: even_count += 1 print("تعداد زوج‌ها:", even_count)
خروجی
تعداد زوج‌ها: 3

جستجو در لیست

پیدا کردن عضو
names = ["علی", "سارا", "رضا", "نیما"] search = "سارا" found = False for name in names: if name == search: found = True break if found: print("پیدا شد!") else: print("پیدا نشد")
پیدا کردن موقعیت
names = ["علی", "سارا", "رضا"] search = "رضا" position = -1 for i, name in enumerate(names): if name == search: position = i break if position != -1: print(f"در ایندکس {position}")

break: از break برای خروج زودهنگام از حلقه استفاده می‌کنیم. وقتی عضو رو پیدا کردیم، دیگه نیازی به ادامه نیست.

ساختن لیست جدید از روی لیست موجود

فیلتر کردن
numbers = [1, 2, 3, 4, 5, 6] evens = [] for num in numbers: if num % 2 == 0: evens.append(num) print("زوج‌ها:", evens)
خروجی
زوج‌ها: [2, 4, 6]
تغییر همه اعضا
numbers = [1, 2, 3, 4] squared = [] for num in numbers: squared.append(num ** 2) print("توان‌ها:", squared)
خروجی
توان‌ها: [1, 4, 9, 16]

پیمایش با حلقه while

می‌تونیم با while هم لیست رو پیمایش کنیم، مخصوصاً وقتی به ایندکس نیاز داریم.

fruits = ["سیب", "موز", "پرتقال"] i = 0 while i < len(fruits): print(f"عضو {i}: {fruits[i]}") i += 1
خروجی
عضو 0: سیب
عضو 1: موز
عضو 2: پرتقال

⚠️ خطاهای رایج:

  • تغییر لیست هنگام پیمایش: موقع پیمایش لیست رو تغییر نده (مثلاً حذف عضو)
  • ایندکس خارج از محدوده: وقتی با while پیمایش می‌کنی، حواست به شرط باشه
  • فراموش کردن len: توی range(len(list)) یادت نره len رو استفاده کنی
  • break بی‌جا: استفاده نابجا از break باعث میشه حلقه زودتر از موعد تموم بشه

خلاصه جلسه ۲۳

  • پیمایش ساده: for item in list
  • پیمایش با ایندکس: for i in range(len(list))
  • دسترسی همزمان به ایندکس و مقدار: enumerate()
  • محاسبات روی لیست: مجموع، میانگین، ماکزیمم
  • جستجو در لیست با حلقه و break
  • ساختن لیست جدید با فیلتر کردن
  • پیمایش با while

تمرینات جلسه ۲۳

تمرین ۱ (ساده):
یه لیست از ۵ اسم بساز و با حلقه for همه رو چاپ کن.
تمرین ۲ (ساده):
با استفاده از enumerate، لیست ["پایتون", "جاوا", "سی"] رو پیمایش کن و به این شکل چاپ کن:
0: پایتون
1: جاوا
2: سی
تمرین ۳ (متوسط):
یه لیست از اعداد ۱ تا ۱۰ بساز. مجموع اعداد رو حساب کن و میانگین رو چاپ کن.
تمرین ۴ (متوسط):
توی لیست [12, 45, 78, 23, 56, 89] بزرگترین عدد رو پیدا کن.
تمرین ۵ (چالشی):
از کاربر ۵ اسم بگیر و توی لیست ذخیره کن. بعد اسم خاصی رو از کاربر بگیر و بگو توی لیست هست یا نه. اگر هست، ایندکسش رو هم بگو.
تمرین ۶ (چالشی):
لیست [2, 7, 4, 9, 3, 6] رو پیمایش کن و یه لیست جدید از اعداد فرد بساز.
تمرین ۷ (پروژه):
یه برنامه مدیریت نمرات بنویس که:
• ۵ نمره از کاربر بگیره و توی لیست ذخیره کنه
• مجموع نمرات رو حساب کنه
• میانگین نمرات رو حساب کنه
• بالاترین و پایین‌ترین نمره رو پیدا کنه
• تعداد قبول‌ها (نمره >= 10) رو بشمره

سوال داری؟

اگه توی پیمایش لیست یا پیدا کردن ماکزیمم مشکل داری، حتماً بپرس.
