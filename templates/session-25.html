توابع - مقدماتی (Functions - Introduction)

به جلسه بیست و پنجم خوش اومدی! تا الان برنامه‌هامون رو به صورت خط به خط می‌نوشتیم. اما توی دنیای واقعی، برنامه‌ها خیلی بزرگتر از این حرفان.

توابع (Functions) به ما کمک می‌کنن کدمون رو تکه‌تکه کنیم، هر تکه یه کار مشخص انجام بده، و هر جا نیاز داشتیم ازش استفاده کنیم.

بردیا فاتحی

تابع یعنی چی؟

تابع یه بلوک از کد هست که یه کار مشخص رو انجام میده. مثل یه دستگاه می‌مونه که بهش یه چیزی میدی، یه کاری روش انجام میده، و بهت برمی‌گردونه.

مثال روزمره: دستگاه قهوه‌ساز رو در نظر بگیر. بهش آب و قهوه میدی، برات قهوه درست میکنه.
توی برنامه‌نویسی: def make_coffee(water, coffee):

✅ چرا از تابع استفاده می‌کنیم؟

  • تکرار نکردن کد (DRY - Don't Repeat Yourself): یه بار می‌نویسی، چند بار استفاده می‌کنی
  • خواناتر شدن کد: کد به بخش‌های کوچیک و مشخص تقسیم میشه
  • دیباگ آسان‌تر: اگه مشکلی پیش بیاد، فقط یه تابع رو چک می‌کنی
  • قابلیت استفاده مجدد: می‌تونی تابع رو توی برنامه‌های دیگه هم استفاده کنی

ساختار تابع در پایتون

توی پایتون، تابع با کلمه کلیدی def تعریف میشه.

def نام_تابع(): # بدنه تابع - کاری که باید انجام بده print("سلام از تابع")

اجزای تابع:

  • کلمه def
  • نام تابع (خودت انتخاب می‌کنی)
  • پرانتز ( )
  • دو نقطه :
  • بدنه تابع با indent

نکات مهم:

  • اسم تابع باید معنی‌دار باشه
  • بعد از def و نام تابع، حتماً پرانتز بذار
  • دو نقطه رو فراموش نکن
  • بدنه تابع باید indent داشته باشه

اولین تابع ما

بیا یه تابع ساده بنویسیم که فقط یه پیغام چاپ کنه.

# تعریف تابع def say_hello(): print("سلام! به دنیای توابع خوش اومدی") print("این اولین تابع منه") # صدا زدن تابع say_hello()
خروجی
سلام! به دنیای توابع خوش اومدی
این اولین تابع منه

توضیح خط به خط:
• def say_hello(): - تابعی به اسم say_hello تعریف می‌کنیم
• print(...) - بدنه تابع (دو خط)
• say_hello() - تابع رو صدا می‌زنیم تا اجرا بشه

تابع با چند خط

توابع می‌تونن چند خط کد داشته باشن. هر چی توی بدنه تابع با indent نوشته بشه، جزو اون تابع محسوب میشه.

def show_info(): print("*" * 30) print("نام: بردیا فاتحی") print("شغل: مدرس برنامه‌نویسی") print("*" * 30) show_info()
خروجی
******************************
نام: بردیا فاتحی
شغل: مدرس برنامه‌نویسی
******************************

توضیح: تابع show_info چهار تا print داره. وقتی تابع رو صدا می‌زنیم، هر چهار خط اجرا میشن.

چند بار صدا زدن تابع

یکی از مهم‌ترین مزایای تابع اینه که می‌تونی چند بار ازش استفاده کنی.

def welcome(): print("به آکادمی خوش آمدید!") # سه بار صدا می‌زنیم welcome() welcome() welcome()
خروجی
به آکادمی خوش آمدید!
به آکادمی خوش آمدید!
به آکادمی خوش آمدید!

توضیح: اگه سه بار تابع رو صدا بزنیم، سه بار اجرا میشه. بدون تابع باید سه بار کد رو می‌نوشتیم.

مثال کاربردی: منوی رستوران

فرض کن می‌خوایم یه برنامه برای رستوران بنویسیم که منو رو نشون بده.

def show_menu(): print("\n" + "="*30) print("منوی رستوران") print("="*30) print("1. پیتزا - 120,000 تومان") print("2. برگر - 80,000 تومان") print("3. سالاد - 50,000 تومان") print("4. نوشابه - 15,000 تومان") print("="*30) # نمایش منو show_menu()
خروجی
==============================
منوی رستوران
==============================
1. پیتزا - 120,000 تومان
2. برگر - 80,000 تومان
3. سالاد - 50,000 تومان
4. نوشابه - 15,000 تومان
==============================

توضیح: ما تابع show_menu رو ساختیم که منو رو چاپ می‌کنه. هر وقت نیاز به نمایش منو داشتیم، فقط کافیه این تابع رو صدا بزنیم.

صدا زدن تابع داخل حلقه

می‌تونیم تابع رو داخل حلقه هم صدا بزنیم.

def count_down(): print("۳") print("۲") print("۱") print("بریم!") for i in range(3): count_down() print("-"*10)

⚠️ خطاهای رایج:

  • فراموش کردن پرانتز: say_hello (بدون پرانتز) تابع رو صدا نمیزنه، فقط به خود تابع اشاره میکنه
  • فراموش کردن دو نقطه: بعد از def باید : بذاری
  • indent اشتباه: کدهای داخل تابع باید فاصله داشته باشن
  • تعریف تابع بعد از صدا زدن: باید اول تابع رو تعریف کنی، بعد صدا بزنی

خلاصه جلسه ۲۵

  • تابع یه بلوک کد برای انجام یه کار مشخص
  • تعریف تابع: def name():
  • بدنه تابع باید indent داشته باشه
  • صدا زدن تابع: name()
  • توابع باعث میشن کد خواناتر، قابل استفاده مجدد و سازماندهی‌شده‌تر بشه
  • می‌تونیم تابع رو چند بار صدا بزنیم
  • می‌تونیم تابع رو داخل حلقه صدا بزنیم

تمرینات جلسه ۲۵

تمرین ۱ (ساده):
تابعی به اسم say_name بنویس که اسمت رو چاپ کنه. بعد تابع رو صدا بزن.
تمرین ۲ (ساده):
تابعی بنویس که ۳ تا از علاقه‌مندی‌هات رو هر کدام توی یه خط چاپ کنه.
تمرین ۳ (متوسط):
تابعی بنویس که یه خط جداکننده (مثلاً ۴۰ تا =) چاپ کنه. بعد ۵ بار تابع رو توی یه حلقه صدا بزن.
تمرین ۴ (متوسط):
تابعی به اسم show_time بنویس که ساعت و تاریخ فعلی رو چاپ کنه. (میتونی از import datetime استفاده کنی)
تمرین ۵ (چالشی):
یه تابع برای نمایش منوی رستوران بنویس (مثل همونی که توی مثال زدیم). بعد یه حلقه while بذار که تا وقتی کاربر گزینه خروج رو انتخاب نکرده، منو رو نشون بده.
تمرین ۶ (پروژه):
یه برنامه ماشین حساب ساده با توابع بنویس که:
• تابعی برای نمایش منو
• تابعی برای جمع
• تابعی برای تفریق
• تابعی برای ضرب
• تابعی برای تقسیم
کاربر از منو انتخاب کنه و برنامه تا وقتی ادامه پیدا کنه که کاربر گزینه خروج رو انتخاب کنه.

سوال داری؟

اگه مفهوم تابع یا نحوه تعریفش رو متوجه نشدی، حتماً بپرس.
