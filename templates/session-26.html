پارامتر و return - ورودی و خروجی توابع

به جلسه بیست و ششم خوش اومدی! توی جلسه قبل با توابع ساده آشنا شدیم که فقط یه کاری انجام می‌دادن. اما توابع واقعی معمولاً ورودی می‌گیرن و خروجی می‌دن.

مثلاً تابع جمع، دو عدد می‌گیره و نتیجه رو برمی‌گردونه. اینجاست که پارامتر (parameter) و return به کمکمون میان.

بردیا فاتحی

پارامتر (Parameter) یعنی چی؟

پارامتر ورودی‌هایی هستن که به تابع می‌دیم تا باهاشون کار کنه.

مثال روزمره: دستگاه قهوه‌ساز رو در نظر بگیر. بهش آب و قهوه می‌دی (پارامتر)، برات قهوه درست می‌کنه.
توی برنامه‌نویسی: def make_coffee(water, coffee):

def greet(name): print("سلام", name, "عزیز!") greet("علی") greet("سارا")
خروجی
سلام علی عزیز!
سلام سارا عزیز!

توضیح خط به خط:
• def greet(name): - تابعی با یه پارامتر به اسم name
• print("سلام", name, "عزیز!") - از مقدار name توی بدنه استفاده می‌کنیم
• greet("علی") - تابع رو با مقدار "علی" صدا می‌زنیم
• greet("سارا") - دوباره با مقدار "سارا" صدا می‌زنیم

توابع با چند پارامتر

یه تابع می‌تونه چندین پارامتر داشته باشه.

def add(a, b): result = a + b print(f"{a} + {b} = {result}") add(5, 3) add(10, 7)
خروجی
5 + 3 = 8
10 + 7 = 17
def show_person(name, age, city): print(f"نام: {name}") print(f"سن: {age}") print(f"شهر: {city}") print("-"*20) show_person("علی", 25, "تهران") show_person("سارا", 30, "اصفهان")
خروجی
نام: علی
سن: 25
شهر: تهران
--------------------
نام: سارا
سن: 30
شهر: اصفهان
--------------------

نکته: ترتیب پارامترها مهمه. وقتی تابع رو صدا می‌زنی، باید مقادیر رو به همون ترتیب بدی.

return - برگردوندن مقدار

return به تابع می‌گه یه مقدار رو برگردونه تا بتونیم توی بقیه برنامه استفاده‌اش کنیم.

def add(a, b): result = a + b return result sum_result = add(5, 3) print("حاصل جمع:", sum_result) print("حاصل ضرب:", sum_result * 2)
خروجی
حاصل جمع: 8
حاصل ضرب: 16

توضیح خط به خط:
• result = a + b - جمع رو توی متغیر result می‌ریزیم
• return result - مقدار result رو برمی‌گردونیم به جایی که تابع صدا زده شده
• sum_result = add(5, 3) - تابع صدا زده میشه، مقدار برگشتی توی sum_result می‌ریزه
• بعدش می‌تونیم با sum_result هر کاری خواستیم بکنیم

تفاوت return و print

خیلی مهم: فرق return و print رو باید خوب بفهمی.

print

فقط توی صفحه نمایش نشون میده

def show(x):
    print(x)

result = show(5)
print(result) # None

return

مقدار رو برمی‌گردونه برای استفاده بعدی

def get(x):
    return x * 2

result = get(5)
print(result) # 10

توضیح: تابعی که فقط print داره، هیچ مقداری برنمی‌گردونه. وقتی result = show(5) می‌کنیم، result میشه None. اما تابع با return یه مقدار برمی‌گردونه که می‌تونیم توی متغیر ذخیره‌اش کنیم.

مثال‌های کاربردی با return

# تابع محاسبه مساحت مستطیل def rectangle_area(length, width): area = length * width return area area1 = rectangle_area(5, 3) area2 = rectangle_area(10, 4) print("مساحت اول:", area1) print("مساحت دوم:", area2)
خروجی
مساحت اول: 15
مساحت دوم: 40
# تابع تبدیل سلسیوس به فارنهایت def celsius_to_fahrenheit(celsius): fahrenheit = (celsius * 9/5) + 32 return fahrenheit temp = celsius_to_fahrenheit(30) print("۳۰ درجه سلسیوس =", temp, "فارنهایت")
خروجی
۳۰ درجه سلسیوس = 86.0 فارنهایت

چند return در یک تابع

می‌تونیم توی یه تابع چند تا return داشته باشیم، اما فقط اولین return که بهش می‌رسیم اجرا میشه و تابع تموم میشه.

def check_number(num): if num > 0: return "مثبت" elif num < 0: return "منفی" else: return "صفر" print(check_number(10)) print(check_number(-5)) print(check_number(0))
خروجی
مثبت
منفی
صفر

توضیح: هر کدوم از شرط‌ها که true باشه، return مربوط به خودش اجرا میشه و تابع تموم میشه.

مقدار پیش‌فرض پارامترها (Default Values)

می‌تونیم به پارامترها مقدار پیش‌فرض بدیم. اگه کاربر مقداری نده، از این مقدار استفاده میشه.

def greet(name, msg="سلام"): print(f"{msg} {name}") greet("علی") greet("سارا", "خوش آمدی")
خروجی
سلام علی
خوش آمدی سارا

نکته: پارامترهای با مقدار پیش‌فرض باید بعد از پارامترهای بدون پیش‌فرض بیان.

⚠️ خطاهای رایج:

  • فراموش کردن return: تابعی که return نداره، None برمی‌گردونه
  • تعداد پارامترها: اگه تعداد آرگومان‌ها با پارامترها یکی نباشه، خطا می‌گیری
  • کد بعد از return: بعد از return هیچ کدی اجرا نمیشه
  • return با پرانتز: return یه دستوره، نه تابع. پس return (x) غلطه، return x درسته

خلاصه جلسه ۲۶

  • پارامتر = ورودی‌های تابع
  • return = خروجی تابع (مقدار برگشتی)
  • فرق print و return: print فقط نمایش میده، return مقدار برمی‌گردونه
  • می‌تونیم چند پارامتر داشته باشیم
  • می‌تونیم چند return داشته باشیم (اما فقط اولی اجرا میشه)
  • پارامترها می‌تونن مقدار پیش‌فرض داشته باشن

تمرینات جلسه ۲۶

تمرین ۱ (ساده):

تمرین ۲ (ساده):
تابعی بنویس که سه عدد بگیره و جمعشون رو برگردونه.
تمرین ۳ (متوسط):
تابعی به اسم is_even بنویس که یه عدد بگیره و اگه زوج بود True وگرنه False برگردونه.
تمرین ۴ (متوسط):
تابعی بنویس که دو عدد و یه عملگر (+, -, *, /) بگیره و نتیجه رو برگردونه.
تمرین ۵ (چالشی):
تابعی به اسم max_of_three بنویس که سه عدد بگیره و بزرگترینشون رو برگردونه.
تمرین ۶ (پروژه):
یه برنامه ماشین حساب با توابع بنویس که:
• تابع جمع (دو عدد می‌گیره، نتیجه برمی‌گردونه)
• تابع تفریق
• تابع ضرب
• تابع تقسیم
• تابع اصلی که منو رو نشون میده و از کاربر ورودی می‌گیره و توابع مناسب رو صدا می‌زنه
