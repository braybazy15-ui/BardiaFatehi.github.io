if-elif-else - تصمیم‌گیری چند راهی

به جلسه پانزدهم خوش اومدی! توی دو جلسه قبل با if و if-else آشنا شدیم. اما توی دنیای واقعی، گاهی بیشتر از دو حالت داریم.

مثلاً: نمرات دانشجو می‌تونه عالی، خوب، متوسط یا ضعیف باشه. اینجاست که if-elif-else به کمکمون میاد.

بردیا فاتحی

مقایسه if، if-else و if-elif-else

if ساده

یک شرط، یک مسیر

if x > 0:
    print("مثبت")

if-else

یک شرط، دو مسیر

if x > 0:
    print("مثبت")
else:
    print("منفی")

if-elif-else

چند شرط، چند مسیر

if x > 0:
    print("مثبت")
elif x < 0:
    print("منفی")
else:
    print("صفر")

if شرط۱: # اگه شرط۱ درست بود print("شرط۱ برقرار") elif شرط۲: # اگه شرط۱ غلط بود و شرط۲ درست print("شرط۲ برقرار") elif شرط۳: # اگه شرط۱ و شرط۲ غلط بودن و شرط۳ درست print("شرط۳ برقرار") else: # اگه هیچ‌کدوم از شرط‌ها درست نبود print("هیچ شرطی برقرار نیست")

نکته خیلی مهم: شرط‌ها به ترتیب بررسی میشن. به محض اینکه یکی از شرط‌ها True شد، بقیه شرط‌ها بررسی نمیشن!

مثال ۱: تبدیل نمره به حرف

score = float(input("نمره خود را وارد کن: ")) if score >= 90: print("نمره: A (عالی)") elif score >= 80: print("نمره: B (خوب)") elif score >= 70: print("نمره: C (متوسط)") elif score >= 60: print("نمره: D (ضعیف)") else: print("نمره: F (مردود)")
خروجی‌ها
نمره ۹۵ → A (عالی)
نمره ۸۵ → B (خوب)
نمره ۷۵ → C (متوسط)
نمره ۶۵ → D (ضعیف)
نمره ۵۰ → F (مردود)

چرا اینجوری کار میکنه؟ مثلاً نمره ۸۵: اول چک میکنه ۸۵>=۹۰؟ نه → بعد ۸۵>=۸۰؟ بله → پس "خوب" چاپ میشه و دیگه بقیه شرط‌ها چک نمیشن.

مثال ۲: رده سنی

age = int(input("سن: ")) if age < 13: print("کودک") elif age < 18: print("نوجوان") elif age < 65: print("بزرگسال") else: print("سالمند")

مثال ۳: وضعیت هوا

temp = float(input("دما: ")) if temp < 0: print("یخبندان") elif temp < 10: print("سرد") elif temp < 25: print("متوسط") elif temp < 40: print("گرم") else: print("خیلی گرم")

پروژه کوچک: ماشین حساب با انتخاب عملگر

num1 = float(input("عدد اول: ")) num2 = float(input("عدد دوم: ")) op = input("عملگر (+, -, *, /): ") if op == "+": print(f"جمع: {num1 + num2}") elif op == "-": print(f"تفریق: {num1 - num2}") elif op == "*": print(f"ضرب: {num1 * num2}") elif op == "/": if num2 != 0: print(f"تقسیم: {num1 / num2}") else: print("تقسیم بر صفر ممکن نیست") else: print("عملگر نامعتبر")

⚠️ اشتباهات رایج در if-elif-else:

  • ترتیب اشتباه شرط‌ها: مثلاً اول شرط >50 بعد >80 (شرط اول همه رو میگیره)
  • نداشتن else: else اختیاریه، ولی بهتره برای پوشش همه حالات باشه
  • شرط‌های هم‌پوشانی: مراقب باش شرط‌ها روی هم نیفتن
  • فراموش کردن : هم if و هم elif و هم else به : نیاز دارن

خلاصه جلسه ۱۵

  • if-elif-else برای چند شرط استفاده میشه
  • شرط‌ها به ترتیب بررسی میشن
  • به محض پیدا شدن اولین شرط True، بقیه شرط‌ها نادیده گرفته میشن
  • میتونیم هر تعداد elif داشته باشیم
  • else اختیاریه و وقتی اجرا میشه که هیچ شرطی True نباشه
  • ترتیب شرط‌ها خیلی مهمه

تمرینات جلسه ۱۵

تمرین ۱ (ساده):
برنامه‌ای بنویس که نمره کاربر رو بگیره و این پیام‌ها رو چاپ کنه:
نمره >= 90: "عالی"
نمره >= 70: "خوب"
نمره >= 50: "متوسط"
نمره < 50: "ضعیف"
تمرین ۲ (متوسط):
بر اساس سن، رده سنی رو مشخص کن:
0-12: کودک
13-19: نوجوان
20-35: جوان
36-60: میانسال
61+: سالمند
تمرین ۳ (متوسط):
برنامه‌ای بنویس که یک عدد از کاربر بگیره و بگه:
مثبت، منفی یا صفر
تمرین ۴ (پروژه):
یه ماشین حساب پیشرفته با منو بنویس که کاربر بتونه انتخاب کنه:
1. جمع
2. تفریق
3. ضرب
4. تقسیم
سپس دو عدد بگیره و نتیجه رو چاپ کنه.
تمرین ۵ (چالشی):
برنامه‌ای بنویس که سه عدد از کاربر بگیره و بزرگترین رو مشخص کنه. (فقط با if-elif-else، بدون max)
تمرین ۶ (خطایابی):
کد زیر مشکل داره. مشکلش چیه و چطور درستش کنیم؟
x = 50
if x > 10:
    print("بزرگتر از ۱۰")
elif x > 30:
    print("بزرگتر از ۳۰")
else:
    print("کوچکتر")
