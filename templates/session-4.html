نصب VS Code (ویرایشگر کد)

پایتون نصب شد! حالا نیاز به یه محیط داریم که توش کدهامون رو بنویسیم. VS Code بهترین ویرایشگر کد برای برنامه‌نویسی پایتونه. رایگان، سبک و فوق‌العاده قدرتمند.

دانلود VS Code

  • وارد سایت code.visualstudio.com بشو
  • روی دکمه Download for Windows کلیک کن
  • فایل نصب (حدود ۸۰ مگابایت) دانلود میشه
  • فایل رو اجرا کن و مراحل نصب رو برو

نکات نصب

  • تیک "Add to PATH" رو حتماً بزن
  • تیک "Add to Context Menu" رو بزن
  • بقیه گزینه‌ها رو پیش‌فرض بذار
  • بعد نصب، VS Code رو باز کن
